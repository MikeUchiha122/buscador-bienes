## 3. ¿QUÉ HERRAMIENTAS USA? (FRAMEWORKS)

### 🎨 Tkinter (pronounced "tee-kay-inter")
**¿Qué es?** Es como los bloques de construcción LEGO para hacer ventanas.

**Analogía:** Si construir una casa es como hacer un programa, Tkinter son las paredes, ventanas y puertas que puedes ver.

**¿Para qué sirve?** Para crear la interfaz visual (botones, tablas, campos de texto).

### 🐼 Pandas (pronounced "pan-das")
**¿Qué es?** Es como una hoja de cálculo muito inteligente en el código.

**Analogía:** Imagina que tienes un Excel muito poderosa que puede hacer operaciones matemáticas, filtros y búsquedas instantáneas.

**¿Para qué sirve?** Para leer, manipular y guardar archivos Excel.

### 📊 OpenPyXL (pronounced "open-pyx-el")
**¿Qué es?** Es el traductor entre Python y Excel.

**Analogía:** Como un intérprete que traduce español a inglés, OpenPyXL traduce el código de Python al idioma de Excel.

**¿Para qué sirve?** Para guardar archivos en formato Excel (.xlsx).

### 🖥️ Python (pronounced "pay-tón")
**¿Qué es?** El lenguaje de programación principal.

**Analogía:** Es como el idioma que habla la computadora.

### CAPÍTULO 4: CARGAR LOS DATOS (Líneas 57-85)

```python
def cargar_datos(self):
    try:
        import numpy as np
        log_accion("INICIO", f"Cargando Excel: {EXCEL_PATH}")
        df = pd.read_excel(EXCEL_PATH, header=0, engine='openpyxl')
        df.columns = df.columns.str.strip()
        
        df = df.loc[:, ~df.columns.isna()]
        
        for col in df.columns:
            if col and isinstance(col, str) and 'INVENTARIO' in col.upper():
                df[col] = df[col].fillna(0).astype(np.int64)
                break
        
        for col in df.columns:
            if df[col].dtype == 'object':
                df[col] = df[col].fillna('').astype(str).str.strip()
        
        if 'STATUS' not in df.columns:
            df['STATUS'] = 1
            df.to_excel(EXCEL_PATH, index=False, engine='openpyxl')
            log_accion("NUEVA_COLUMNA", "Se agregó columna STATUS al Excel")
        
        log_accion("CARGA_EXITO", f"Total bienes cargados: {len(df)}")
        return df
    except Exception as e:
        log_accion("ERROR_CARGA", str(e))
        messagebox.showerror("Error", f"No se pudo cargar el Excel:\n{e}")
        return pd.DataFrame()
```

**Explicación paso a paso:**

1. **try:** = Vamos a intentar hacer algo que podría fallar
2. **pd.read_excel(...)** = Abrir el archivo Excel y leerlo
3. **df.columns.str.strip()** = Limpiar los nombres de las columnas (quitar espacios)
4. **df.loc[:, ~df.columns.isna()]** = Eliminar columnas vacías
5. **for col in...** = Buscar la columna de INVENTARIO y asegurarnos que sea número
6. **if 'STATUS' not in...** = Si no existe la columna STATUS, crearla
7. **df.to_excel(...)** = Guardar los cambios en el Excel
8. **except Exception as e:** = Si algo sale mal, mostrar error

**Para un niño de 5 años:** Es como cuando revisas tu cuaderno y arreglas las páginas manchadas antes de empezar a trabajar.

### CAPÍTULO 6: FUNCIÓN DE BÚSQUEDA (Líneas 243-285)

```python
def buscar(self):
    try:
        criterio = self.buscar_var.get().strip().upper().replace(" ", "")
        campo = self.campo_busqueda.get()
        filtro = self.filtro_status.get()
        
        if filtro == "VIGENTES":
            df_filtrado = self.df[self.df['STATUS'] == 1].copy()
        elif filtro == "BAJAS":
            df_filtrado = self.df[self.df['STATUS'] == 0].copy()
        else:
            df_filtrado = self.df.copy()
        
        if not criterio:
            self.llenar_tabla(df_filtrado)
            return
        
        if campo == "Todos":
            mask = df_filtrado.apply(lambda row: criterio in row.astype(str).values, axis=1)
        else:
            campo_en_df = self.encontrar_columna(campo)
            mask = df_filtrado[campo_en_df].astype(str).str.contains(criterio, na=False, regex=False)
        
        resultados = df_filtrado[mask]
        
        if resultados.empty:
            messagebox.showinfo("Sin resultados", f"No se encontraron bienes con: '{criterio}'")
        else:
            self.llenar_tabla(resultados)
    except Exception as e:
        messagebox.showerror("Error", f"Error en búsqueda:\n{e}")
```

**Paso a paso:**
1. Obtener lo que el usuario escribió (criterio)
2. Obtener el campo donde buscar (campo)
3. Obtener el filtro (VIGENTES/BAJAS/TODOS)
4. Aplicar el filtro al dataframe
5. Buscar el texto en los registros
6. Mostrar los resultados en la tabla
7. Si no hay resultados, mostrar mensaje

### CAPÍTULO 8: CRUZAR INVENTARIO (Funciones añadidas)

#### 8.1 Cruzar solo por Inventario

```python
def cruzar_inventario(self):
    try:
        nuevo_path = os.path.join(BASE_DIR, "archivo_cruzar", "ARCHIVO_NUEVO.xlsx")
        
        if not os.path.exists(nuevo_path):
            messagebox.showerror("Error", "No se encuentra el archivo")
            return
        
        df_base = self.df.copy()
        df_nuevo = pd.read_excel(nuevo_path)
        
        col_inv = self.encontrar_columna("N° INVENTARIO")
        df_base[col_inv] = df_base[col_inv].astype(str).str.strip()
        df_nuevo[col_inv] = df_nuevo[col_inv].astype(str).str.strip()
        
        inv_base = set(df_base[col_inv].unique())
        inv_nuevo = set(df_nuevo[col_inv].unique())
        
        existentes = df_nuevo[df_nuevo[col_inv].isin(inv_base)]
        faltantes = df_nuevo[~df_nuevo[col_inv].isin(inv_base)]
        
        output_file = os.path.join(BASE_DIR, "INFORME_CRUZE_INVENTARIO.xlsx")
        
        with pd.ExcelWriter(output_file, engine='openpyxl') as writer:
            existentes.to_excel(writer, sheet_name='Existentes', index=False)
            faltantes.to_excel(writer, sheet_name='Faltantes', index=False)
        
        os.startfile(output_file)
    except Exception as e:
        messagebox.showerror("Error", f"Error al realizar el cruze:\n{e}")
```

**¿Qué hace esta función?**
1. Busca el archivo nuevo en la carpeta "archivo_cruzar"
2. Lee el Excel base y el Excel nuevo
3. Compara solo por NÚMERO DE INVENTARIO
4. Crea un informe con dos hojas:
   - Existentes: Los que están en ambos archivos
   - Faltantes: Los que solo están en el nuevo
